## General
Extracts information from ELF object files.

## -a
all of below

## -h
header

## -l
segment headers

## -S
section headers

## -s
symbols

## -r
relocation section

## -d
dynamic section

## -V
version sections

## -A
architecture-specific information

## -I 
bucket list lengths
